# Proyecto

Diseña un Design System completo para una aplicación móvil llamada "Guía Salud".

## Descripción del producto

Guía Salud es una aplicación mobile orientada a ayudar a personas que presentan síntomas, molestias o problemas de salud y necesitan orientación rápida, clara y segura.

La aplicación realiza:

* Evaluación guiada mediante preguntas cerradas.
* Clasificación de riesgo basada en un sistema de triage de 3 niveles.
* Recomendaciones orientativas.
* Detección de señales de alarma.
* Geolocalización de hospitales, centros de salud y farmacias.
* Traducción contextual de frases médicas para viajeros.
* Asistencia conversacional mediante IA.

La aplicación NO reemplaza atención médica profesional.
